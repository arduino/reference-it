= sq(x)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Calcola il quadrato di un numero: il numero moltiplicato per se stesso.
[%hardbreaks]


[float]
=== Sintassi
`sq(x)`


[float]
=== Parametri
`x`: il numero, qualsiasi tipo

[float]
=== Restituisce
Il quadrato del numero. (double)
