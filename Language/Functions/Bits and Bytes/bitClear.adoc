= bitClear()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Resetta ( imposta a 0 ) un bit di una variabile numerica.
[%hardbreaks]


[float]
=== Sintassi
`bitClear( var, n )`


[float]
=== Parametri
`var`: la variabile numerica di cui resettare il bit

`n`: quale bit resettare, partendo da 0 per il bit meno significativo ( il pi√π a destra )

[float]
=== Restituisce
Nulla
