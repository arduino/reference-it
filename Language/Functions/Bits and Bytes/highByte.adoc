= highByte()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Estrae il byte di ordine superiore (più a sinistra) di una parola (o il secondo byte più basso di un tipo di dati più grande). 
[%hardbreaks]


[float]
=== Sintassi
`highByte(x)`


[float]
=== Parametri
`x`: un valore di ogni tipo

[float]
=== Restituisce
byte
