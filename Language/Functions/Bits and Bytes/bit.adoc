= bit()


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Descrizione
Calcola il valore del bit specificato ( il bit 0 è 1, il bit 1 è 2, il bit 2 è 4, ecc... ).
[%hardbreaks]


[float]
=== Sintassi
`bit(n)`


[float]
=== Parametri
`n`: il bit di cui calcolare il valore

[float]
=== Restituisce
Il valore del bit.

--
