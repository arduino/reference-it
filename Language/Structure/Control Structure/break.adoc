[float]
=== Codice di esempio
Nel codice che segue, il controllo esce dal ciclo `for` quando il valore del sensore supera la soglia.
[source,arduino]
----
for (x = 0; x < 255; x ++)
{
    analogWrite(PWMpin, x);
    sensore = analogRead(sensorPin);
    if (sensore > soglia){      // bail out on sensor detect
       x = 0;
       break;
    }
    delay(50);
}
----
